DevOps/DevSecOps CI/CD PIPELINE:

    Git -> Jenkins -> Terraform -> Packer

Git: Version Control
Jenkins: CI Pipeline
Blue Ocean: CD Pipeline
Terraform: Architecture Template
Packer: AWS AMI Build

Steps:
    1: git clone app repo
    2: packer build
    3: git clone terraform repo
    4: terraform apply

Terraform Commands:
    1. terraform init
    2. terraform plan
    3. terraform validate
    4. terraform apply
